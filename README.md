# Project-Correct

- 一個基於React與firebase運用的小型專案。
- 專案發想來自語言交換網站，故主要功能為會員系統、留言回覆功能，以及可以針對單一回覆做訂正修改的功能
- [專案網址](https://project-correct.web.app/)
